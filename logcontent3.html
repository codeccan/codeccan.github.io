    2018-01-07【转载】高效jquery（一）

    您现在的位置是:首页 > 个人日志

    jquery是现在最常用的js框架了，非常简便，但是代码的质量也是有高有低，如何高质量的写出代码，下面总结一下，一下内容有来源于网络搜集。

    1、选择捷径

    // 糟糕

    if(collection.length > 0){..}

    // 建议

    if(collection.length){..}

    2、熟记技巧

    // 糟糕

    $('#id').data(key,value);

    // 建议

    $.data('#id',key,value);

    3、精简javascript

    // 糟糕

    $first.click(function(){

    $first.css('border','1px solid red');

    $first.css('color','blue');

    });

    // 建议

    $first.on('click',function(){

    $first.css({

    'border':'1px solid red',

    'color':'blue'

    });

    });

    4、避免多个id选择符

    // 糟糕

    $('#div #con');

    // 建议

    $('#con');

    5、优化选择符

    // 糟糕

    $('div#div');

    $('div#div a.address');

    // 建议

    $('#div');

    $('#div .address');
