    2018-01-07【转载】高效jquery（二）

    您现在的位置是:首页 > 个人日志

    1、选择捷径

    // 糟糕

    d = $('#ele').height();

    $('#ele').css('height',d-20);

    // 建议

    $ele = $('#ele');

    d = $ele.height();

    $ele.css('height',d-20)

    2、避免全局变量

    // 糟糕

    $ele = $('#ele');

    d = $ele.height();

    $ele.css('height',d-20);

    $('#ele').css('height',d-20);

    // 建议

    var $ele = $('#ele');

    var d = $ele.height();

    $ele.css('height',d-20);

    3、链式操作

    // 糟糕

    $second.html(value);

    $second.on('click',function(){

    alert('hello everybody');

    });

    $second.fadeIn('slow');

    $second.animate({height:'120px'},500);

    // 建议

    $second.html(value);

    $second.on('click',function(){

    alert('hello everybody');

    }).fadeIn('slow').animate({height:'120px'},500);

    4、维持代码可读性

    伴随着精简代码和使用链式的同时，可能带来代码的难以阅读。添加缩紧和换行能起到很好的效果。

    5、利用CDN

    CND能保证选择离用户最近的缓存并迅速响应。
